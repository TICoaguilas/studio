# **App Name**: ClockWise

## Core Features:

- User Authentication: Allow users to select their names from a pre-defined list for clocking in/out.
- Clock In/Out Button: A single button that toggles between 'Clock In' and 'Clock Out' based on the user's current status.
- Timestamp Recording: Record the clock in and clock out times, associated with the user. The information must be stored into a database for persistence
- Data logging: Each event will log the current IP address, so malicious attempts from users sharing accounts could be detected.
- Admin Interface: An admin interface will list user activity.
- Clock-in and Clock-out reports: List data to admins in order to ease closing the current week, month or any specific ranges of dates. Format can be exported into standard spreadsheet format (CSV or similar).

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean, uncluttered interface.
- Accent color: Teal (#009688) for clock-in/out button to draw attention.
- Font: 'Inter' (sans-serif) for both headlines and body text, ensuring readability and a modern feel.
- Use simple, clear icons to represent clock in/out actions.
- Clean and intuitive layout with clear labeling and user-friendly navigation.
- Subtle transitions when clocking in/out to provide visual feedback.
